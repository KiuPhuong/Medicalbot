# Medicalbot

### clone the repository
project repo: https://github.com/KiuPhuong/Medicalbot.git

## Setup Environment
### STEP 1
```bash
conda create -n medibot python=3.11.0 -y
```

```bash
conda activate medibot
```

## Install the requirements
### STEP 2
```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```
